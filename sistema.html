tokens.css / design system 
:root {
  /* Backgrounds */
  --onyx:      #08090C;   /* fondo principal     */
  --carbon:    #13141A;   /* fondo secundario    */
  --smoke:     #2A2B33;   /* bordes, separadores */

  /* Accents */
  --champagne: #C8A876;   /* acento, CTAs        */
  --ice:       #6FB8FF;   /* acento eléctrico    */

  /* Text */
  --snow:      #F4F4F6;   /* texto principal     */
  --mist:      #8B8B95;   /* texto secundario    */
}

03 / COMPONENT

Botón CTA primario

Outline en reposo, fill on commit. Animación con easing humano.

button.css 
.btn-send {
  background:     transparent;
  border:         1.5px solid var(--champagne);
  color:          var(--champagne);
  padding:        18px 32px;
  letter-spacing: 0.18em;
  transition:     all 0.4s var(--ease);
}

.btn-send:hover,
.btn-send--ready {
  background: var(--champagne);
  color:      var(--onyx);
}
